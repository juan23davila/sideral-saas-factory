# SaaS Factory - CLAUDE.md

Este archivo define las reglas de operación para el agente de IA (Antigravity/Claude Code) dentro de este proyecto.

## Estándares del Proyecto
- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS
- **Components**: Lucide React + Componentes Premium (Glassmorphism)
- **Language**: TypeScript

## Comandos de Generación de Código
Para generar código base, el usuario puede usar comandos como:

### `setup-base`
Genera la estructura inicial de la aplicación, incluyendo:
- `src/app/layout.tsx`: Layout con diseño premium.
- `src/app/page.tsx`: Landing page de bienvenida.
- `src/components/ui`: Componentes base.

### `generate-page [nombre]`
Crea una nueva página en el App Router con diseño responsivo y premium.

## Reglas de Estilo
- Usar gradientes modernos (vibrant colors).
- Implementar Glassmorphism por defecto en tarjetas y paneles.
- Asegurar que todos los componentes sean responsivos.
