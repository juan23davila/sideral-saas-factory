# SaaS Factory - GEMINI.md

Este archivo proporciona instrucciones persistentes para el agente Gemini CLI.

## Contexto del Proyecto
Estás trabajando en una "SaaS Factory" diseñada por Juan Davila. El objetivo es crear aplicaciones web de alta calidad rápidamente.

## Instrucciones de Desarrollo
- Prioriza la velocidad de ejecución y la calidad visual.
- Usa bibliotecas estándar para evitar dependencias innecesarias.
- Sigue el patrón de App Router de Next.js.
